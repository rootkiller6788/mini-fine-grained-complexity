## External Prerequisites

```
Classical Complexity Theory (required background):
  - P, NP, NP-completeness
  - Polynomial-time reductions (Karp, Cook)
  - Time hierarchy theorem
  - Boolean circuits

Algorithm Design (required background):
  - Dynamic programming (Floyd-Warshall, Edit Distance, LCS)
  - Sorting and two-pointer technique (3SUM)
  - Bit manipulation (OV)
  - Matrix multiplication

Graph Theory (helpful but not required):
  - Shortest paths
  - Graph metrics (radius, diameter, median)
  - Adjacency matrices

Computational Geometry (helpful):
  - Point-line duality (3SUM -> Collinearity)
  - Frechet distance
  - Polygonal curves
```
