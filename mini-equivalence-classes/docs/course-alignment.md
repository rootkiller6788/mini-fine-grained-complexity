## Learning Path

1. **Foundations**: Classical complexity (P, NP, polynomial-time reductions)
2. **SETH & ETH**: Exponential Time Hypothesis, Strong ETH
3. **Subcubic Equivalence**: APSP, Min-Plus, Negative Triangle
4. **Subquadratic Equivalence**: OV, Edit Distance, LCS, DTW
5. **3SUM Equivalence**: 3SUM, collinearity, geometric problems
6. **Cross-Class Relations**: How classes relate, open problems
