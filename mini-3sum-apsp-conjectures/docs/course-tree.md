# Course Tree: 3SUM and APSP Conjectures\n\n## Prerequisites\n\n### Required Knowledge\n- Basic algorithm analysis (big-O notation)\n- Sorting algorithms (quicksort, binary search)\n- Graph algorithms (shortest paths, BFS/DFS)\n- Matrix operations\n- Hash tables\n\n### From mini-complexity-foundations\n- P, NP definitions (complexity classes)\n- Polynomial-time reductions\n- Cook-Levin theorem (NP-completeness concept)\n\n### From mini-fine-grained-complexity\n- mini-seth-strong-eth: SETH/ETH definitions\n- mini-orthogonal-vectors: OV problem and conjecture\n- mini-k-clique-hardness: conditional lower bounds\n\n## Dependency Tree\n\n```\nAlgorithm Analysis Basics\n    |\n    +--> Sorting (O(n log n))\n    |       |\n    |       +--> 3SUM: sort+bsearch, two-pointer\n    |\n    +--> Graph Algorithms\n    |       |\n    |       +--> Shortest Paths (Dijkstra, Bellman-Ford)\n    |       |       |\n    |       |       +--> APSP: Floyd-Warshall, Johnson\n    |       |\n    |       +--> Graph Theory (radius, diameter, centrality)\n    |\n    +--> Matrix Operations\n    |       |\n    |       +--> Min-Plus Product\n    |       |       |\n    |       |       +--> APSP via repeated squaring\n    |       |\n    |       +--> Strassen Algorithm\n    |               |\n    |               +--> Strassen-inspired min-plus\n    |\n    +--> Hash Tables\n    |       |\n    |       +--> Hash-based 3SUM O(n^2)\n    |\n    +--> Complexity Theory\n            |\n            +--> Polynomial-time reductions\n            |       |\n            |       +--> Fine-grained reductions\n            |               |\n            |               +--> 3SUM <-> APSP equivalence\n            |\n            +--> Conditional lower bounds\n                    |\n                    +--> 3SUM/APSP conjectures\n```\n\n## Learning Path\n\n1. Understand 3SUM problem and naive O(n^3) solution\n2. Learn two-pointer technique -> O(n^2) 3SUM\n3. Understand APSP and Floyd-Warshall O(n^3)\n4. Learn Johnson algorithm for sparse graphs\n5. Study min-plus semiring and matrix formulation\n6. Understand fine-grained reductions\n7. Study VW 2009 equivalence theorem\n8. Explore equivalence hexagon (Williams & Williams 2010)\n9. Learn Gronlund-Pettie 2014 breakthrough ideas\n10. Explore research frontiers\n\n## Recommended Reading Order\n1. Gajentaan & Overmars (1995) - foundational\n2.
